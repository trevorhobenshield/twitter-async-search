Fast twitter search using undocumented API

![](assets/example.gif)

```bash
pip install twitter-async-search
```

### Example
```bash
twitter-async-search 'query1' 'query2' 
```
Search results are output to `~/data/raw`

### Notes
- ~400 search results rate limiting occurs
